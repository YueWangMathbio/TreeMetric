Code for manuscript "Two Metrics on Rooted Unordered Trees with Labels" by Yue Wang, https://arxiv.org/abs/2103.11553.

BM.py: Python code for calculating the best-match metric D_BM for two unordered rooted trees with possibly repeated labels.

LR.py: Python code for calculating the left-regular metric D_LR for two unordered rooted trees with possibly repeated labels.
